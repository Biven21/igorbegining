рисуем мужчину, относительный центр координат находится в точке соприкосновения головы и туловища.

Parameters
x- координата по горизонтальной оси точки отсчета мужчины,
y- координатa по оси y (направлена вниз),
sizeX- размер масштабирования по оси х,
sizeY- размер масштабирования по оси Y,
shovelUp- параметр отвечает за поднятие лопаты 1 - лопата внизу (-1) лопата поднята вверх ,
LeftKneeX- параметр отвечает за расположение левого колена по оси Х,
LeftKneeY- параметр отвечает за расположение левого колена по оси Y,
LeftFootX- параметр отвечает за расположение левой стопы по оси Х
LeftFootY- параметр отвечает за расположение левой стопы по оси Y
RightKneeX- параметр отвечает за расположение правого колена по оси Х,
RightKneeY- параметр отвечает за расположение правого колена по оси Y,
RightFootX- параметр отвечает за расположение правой стопы по оси Х
RightFootY- параметр отвечает за расположение правой стопы по оси Y
Note
мужчина пока с прямыми руками, но планируются локти с возможностями копать лопатой. 
   Особенности движения ног при анимации:
   Базовое движение - левое колено вверху, правое колено внизу и наооборот.
   чтобы сделать еще фазу движения - когда ноги вместе - это когда надо показать человека стоящего,
   необходимо сделать смещение фазы подъема колен на какую либо величину (подбирается индивидуально)
   в предлагаемом примере величина смещения равна 2.
   пример кода будет выглядеть следующим образом:
